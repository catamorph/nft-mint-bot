# 🏎 Nft mint bot 🏎

A minting bot to mint nft for a given smart contract using multiple wallets for the The Great Mint Race competition.

## Installation

create .env file in project root folder.
NODE_URL
CONTRACT_ADDRESS
FIRST_PRIVATE_KEY
SECOND_PRIVATE_KEY
THIRD_PRIVATE_KEY

Fill in the properties above to get started.

npm install
npm run start:dev
